# Kernel Methods in Machine Learning

*This repo is for the assisgnments of the course Kernel Methods in Machine Learning at Aalto University.*

There are 4 assignments:
* Assignment 1:
  * Gaussian-Kernel using matrix operations
  * Parzen Window Classifier
* Assignment 2:
 * Kernel Ridge Regression
 * Kernel pre-processing: Centering & Normalization
 * Predictions of Fingerprints
* Assignment 3:
 * Support Vector Machines
 * Variants of Kernels
* Assignment 4:
 * Canonical correspondance analysis (With Kernels)
